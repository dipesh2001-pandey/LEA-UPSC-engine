# LEA Composite Theme Template v1.0

## Purpose

A Composite Theme groups together multiple Core Themes into a coherent learning domain.

It does not duplicate the knowledge owned by Core Themes.

Its primary responsibility is to define scope, structure, learning sequence, and relationships.

# Composite Theme Principles

- Composite Themes orchestrate; they do not own detailed knowledge.
- Core Themes remain the single source of truth.
- No duplication between Composite and Core Themes.
- Every Composite Theme owns one Theme Map (Canvas).
- Composite Themes optimize navigation, orientation, and learning flow.
- Simplicity takes precedence over completeness.
