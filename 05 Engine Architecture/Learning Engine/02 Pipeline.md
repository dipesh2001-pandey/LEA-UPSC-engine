# Learning Engine Pipeline v1.0

```text
Learning Need
        │
        ▼
1. Learning Planning
        │
        ▼
2. Understanding
        │
        ▼
3. Internalization
        │
        │
        ├── Knowledge Engine
        │   (Build / Update Theme)
        ▼
4. Integration
        │
        ▼
5. Reinforcement
        │
        ▼
6. Application
        │
        ▼
7. Reflection
        │
        ▼
8. Learning Progress Update
        │
        ▼
Assessment Engine
        │
        ▼
Governance Engine
```

## Stage Responsibilities

### 1. Learning Planning

- Define learning objective.
- Select acquisition strategy.
- Establish learning scope.

### 2. Understanding

- Acquire understanding through lectures, books, discussions, and dialectic learning.

### 3. Internalization

- Transform understanding into durable knowledge.
- Build or update Themes through the Knowledge Engine.

### 4. Integration

- Connect the Theme with existing Themes and broader knowledge.

### 5. Reinforcement

- Strengthen memory through revision, retrieval, and repeated exposure.

### 6. Application

- Apply understanding through PYQs, answer writing, case studies, and real-world contexts.

### 7. Reflection

- Evaluate understanding, identify misconceptions, and discover improvements.

### 8. Learning Progress Update

- Record learning outcomes.
- Identify next actions.
- Prepare for Assessment and Governance.
